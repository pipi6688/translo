# Translo

A lightweight Chrome extension for instant translation. Select any text on a webpage, get translations immediately from multiple sources.

## Features

- **Select & Translate** — Highlight any text on a webpage, translation appears in a floating panel right where you selected
- **Multiple AI Providers** — Supports OpenAI, Anthropic, Google AI, and DeepSeek. Compare translations side by side
- **Google Translate Built-in** — Free Google Translate works out of the box, no API key needed
- **Streaming Output** — AI translations stream in real-time, no waiting for the full response
- **10 Target Languages** — Chinese (Simplified/Traditional), English, Japanese, Korean, French, German, Spanish, Russian, Portuguese
- **Customizable Prompts** — Fine-tune system prompts, translation prompts, temperature, and model for each AI provider
- **Custom Base URL** — Use any OpenAI-compatible API endpoint

## Install

1. Download the latest release from [Releases](https://github.com/pipi6688/translo/releases)
2. Unzip the downloaded file
3. Open Chrome → `chrome://extensions/`
4. Enable **Developer mode** (top right toggle)
5. Click **Load unpacked** → select the unzipped folder

## Usage

1. Select any text on a webpage
2. A floating translation panel appears near your selection
3. Click the **Copy** button to copy the translation

### Configure AI Providers

1. Click the Translo extension icon → **Settings**
2. Expand any AI provider (OpenAI, Anthropic, Google AI, DeepSeek)
3. Enter your API key
4. Click **Test** to verify the connection
5. Set your preferred provider as the default translation source
